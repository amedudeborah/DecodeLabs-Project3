# SQL Business Insights Analysis

## Overview

This project uses SQL to analyze an e-commerce dataset and answer business-related questions. Various SQL queries and functions were applied to extract meaningful insights from the data and support business decision-making.

## Objectives

* Analyze sales and order data
* Answer key business questions
* Identify trends and patterns in customer transactions
* Generate actionable business insights

## Tools Used

* SQL
* SQL Functions
* Aggregate Functions
* Filtering and Sorting Queries
* Group By and Order By Clauses

## Analysis Performed

* Product Performance Analysis
* Revenue Analysis
* Payment Method Analysis
* Order Status Analysis
* Customer Transaction Analysis

## Business Questions Answered

* Which products generated the highest revenue?
* Which payment methods were used most frequently?
* How many orders were delivered, cancelled, or returned?
* Which products had the highest and lowest sales?
* What revenue was generated by each product category?

## Files Included

* SQL Queries.sql
* Dataset
* Project Report.pdf

## Outcome

The SQL queries successfully extracted valuable insights from the dataset, helping to evaluate business performance and support data-driven decision-making.
